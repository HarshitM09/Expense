# Expense Splitter

A simple web app to split expenses among roommates.

## Features

- Manage members/roommates
- Add expenses with splitting
- View expense history
- Calculate balances and settlements

## Tech Stack

- Frontend: React, Tailwind CSS
- Backend: Node.js, Express
- Database: JSON files (for MVP)

## Setup

1. Install backend dependencies: cd server && npm install
2. Install frontend dependencies: cd client && npm install
3. Start backend: cd server && npm start
4. Start frontend: cd client && npm start
5. Open http://localhost:3000
